Desarrollo front-end like a boss aprendiendo Bootstrap 3

En este workshop aprenderas...

En este workshop de 7 horas de duracion Maycon Huayapa te mostrará como optimizar tu flujo de trabajo a la hora de realizar el front-end de un proyecto web. La metodoloía apoyada en el framework Bootstrap 3 parte desde la fase de diseño hasta finalizar en la fase de desarrollo. Aprende´ras a diseñar para bootstrap, a orientar el diseño bajo la filosofía mobile first y respnsive web design y descubrirás la metodolog+ia adecuada para convertir ese diseño en una maqueta HTML5 de una forma ágil y sencilla.

Como siempre podrás completar todo lo que aprendas durante el workshop compartiendo expereiencias con otros asistentes y el/los ponentes durante la comida o la cena típica asturiana que degustarán todos los que se animen.
